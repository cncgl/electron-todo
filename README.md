# How to Use

npm は @2 でインストールしてください。
```
$ npm i -g npm@2
```



ESLint は 1.0.0-rc-3 で確認しています。グローバルインストール推奨です。

```
$ npm i -g eslint@1.0.0-rc-3
```


クローンと node module　のインストール

```
$ git clone https://github.com/cncgl/electron-todo.git
$ npm -i
$ npm run build
$ electron .
```
